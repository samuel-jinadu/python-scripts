# Python Scripts

A collection of Python scripts for learning and personal automation projects.

## About

This repository was created for completing the practice exercises from **Automate the Boring Stuff with Python, 3rd Edition**. It has since evolved to include additional scripts developed for personal use and automation tasks.

## Features

- Python-based automation scripts
- Learning exercises and examples
- Reusable utility functions
- Practical solutions for everyday tasks

## Language

- **Primary Language**: Python 3

## Getting Started

### Prerequisites

- Python 3.7 or higher

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/samuel-jinadu/python-scripts.git
   cd python-scripts
   ```

2. (Optional) Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. Install any required dependencies (if applicable):
   ```bash
   pip install -r requirements.txt
   ```
